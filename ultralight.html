deconstructing abstraction in code

I want to code literally...remove the abstraction and expose the process in full detail. At least what I do on my end.
If code is an encoding, an abstraction to remove the visibility of the programmer from the user, then what is it if my code makes the programmer extremely visible?
Does this lighten the load? Is the load a balance between the producer and the consumer?

does it need to have purpose?

Is creating more value always a goal in making a website? Do I share information?
How can I create a website that lessens the load? That doesn't add but subtracts?
By virtue of reading, of absorbing content and of existing and consuming, do we become heavier? When I share, do I become lighter and the viewer heavier? Is lighter good?
